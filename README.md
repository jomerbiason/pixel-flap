# 🐦 Pixel Flap

A fast-paced **Flappy Bird-style arcade game** built with pure HTML5 Canvas and JavaScript.
Simple mechanics, smooth gameplay, and instantly playable in your browser.

---

## 🎮 Play the Game

👉 https://jomerbiason.github.io/my-pixel-flap/

---

## ✨ Features

* 🎯 Simple one-click controls
* 🧠 High score tracking (saved locally)
* 🎮 Smooth gameplay loop
* 📱 Works on desktop and mobile
* ⚡ No frameworks — pure JavaScript

---

## 🕹 Controls

* **Click / Press any key** → Flap
* **M** → Return to Menu
* **C** → View Controls
